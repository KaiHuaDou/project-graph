## Introduction

Project Graph is a software that focuses on quickly drawing diagrams. It aims to help you brainstorming more efficiently. You can create and connect nodes by simply dragging them.

## Installation

### Download directly

<Download />

<Callout type="warn">
**If you are using macOS**

macOS users may encounter a "Project Graph is damaged and can't be opened" error when they first download and run the app.

Solution:

Right click on the app and select "Open".

Or open Terminal and run `xattr -r -d com.apple.quarantine /Applications/Project\ Graph.app`

[More information](/docs/app/faq/mac)

</Callout>
