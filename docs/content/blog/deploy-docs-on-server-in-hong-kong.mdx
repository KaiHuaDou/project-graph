由于有用户反馈官网访问速度缓慢，经过排查后发现有两个原因：

- Cloudflare 在中国大陆访问本身就慢。
- 由于 Cloudflare Worker 限制应用大小，所有不得不放弃 Next.js 的 SSR 渲染能力，使用静态页面部署。

所以我决定将网站迁移到服务器上。

## 部署网站

我首先想到的是在 Github Actions 中使用 pm2 部署应用。但是，我无意间看到了一个名叫 Coolify 的项目，它可以轻松地在服务器上部署使用各种框架的应用，并且无需手动配置构建和启动命令以及环境。

Coolify 的使用方法很简单，一条命令就可以完成安装。安装完成后，我探索了一下它的功能，一段时间后，也是顺利部署好了网站并配置了域名。
